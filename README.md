# API Valorant NET

---
Pequeño proyecto de .NET framework.
La idea del proyecto es tenerlo siempre en mi Github como una referencia y poder volver aca si es que lo necesito.
---

El proyecto ocupa las siguientes tecnologias:
.NET Framework
Entity Framework
Npgsql plugin para Entity framework
Swagger
---

La base de datos es una base de datos que ya existia y que habia usado en otro proyecto. De hecho la idea de una API de valorant ya la he hecho, pero esta vez la pase de Javascript a C#
